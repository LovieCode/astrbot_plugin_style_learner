# 表达风格学习插件

让你的 AI 机器人学会群友的说话方式，回复时更有"人味"。

## 它能做什么

**自动学习**：机器人会默默观察群里的聊天，记住大家常用的表达方式和梗。

- 比如群里经常说"好耶""蚌埠住了""我哭死"，机器人会记下来
- 下次回复时，它会用类似的语气和词汇，而不是冷冰冰的官方回答
- 还会按情绪分类：开心、嘲讽、无奈、鼓励……在合适的场合用合适的语气

**黑话词典**：自动识别群里的缩写和黑话（比如 yyds、xswl、内卷），并尝试推断含义。

**管理员审核**：学到的内容可以先经过管理员确认再启用。可以在管理页面上点通过或拒绝。

**内容守卫**：LLM 只能通过 `send_message_to_user` 工具发送消息，直接输出的文本会被拦截（防止 LLM "心里话"泄露）。

**防抖**：群聊多条消息连发时，只让最后一条触发 LLM 调用，避免重复回复。

**上下文注入**：可选将最近 N 条聊天记录（含其他群成员）注入 LLM 请求，帮助理解语境。

## 怎么用

### 安装

1. 在 AstrBot 的插件市场搜索"表达风格学习"，点击安装
2. 或者手动把插件文件夹放到 data/plugins/ 下
3. 重启 AstrBot 或重载插件

### 快速开始

装好后什么都不用配，机器人会自动开始学习。你可以：

1. 打开 WebUI 侧边栏 - 插件 - 找到本插件 - 点击"页面"进入管理界面
2. 在表达列表页审核/编辑/删除学到的表达方式
3. 在黑话管理页编辑黑话的含义
4. 在统计页查看学习进度

### 常用指令

在群里发送：

| 指令             | 作用            |
| -------------- | ------------- |
| /style list    | 查看当前学到的表达方式列表 |
| /style list 开心 | 只查看"开心"类的表达   |
| /style stats   | 查看学习统计数据      |

## WebUI 页面说明

管理界面分四个标签页：

### 表达列表

显示所有学到的表达方式，每条包含：

- 情境：在什么情况下使用（例如"群友分享好消息时"）
- 风格：对应的说话方式（例如"好耶！太棒了"）
- 情绪：属于哪种情绪分类
- 学习次数：这条表达被验证了多少次

操作按钮：通过 / 拒绝 / 编辑 / 删除

### 黑话管理

显示识别到的网络用语和缩写，可以编辑它们的含义。

### 统计

展示学习总量和情绪分布，可以手动触发学习。

### 设置

| 设置项                | 作用                                 |
| ------------------ | ---------------------------------- |
| 启用表达学习             | 总开关                                |
| 启用黑话挖掘             | 总开关                                |
| 注入模式               | append=追加到回复末尾，tool=工具调用注入，both=两者 |
| 表达选择模式             | simple=快速省token，classic=更准但费token  |
| 仅用审核通过的表达          | 开启后只使用管理员确认过的内容                    |
| 学习触发阈值             | 攒多少条消息学一次（默认30条）                   |
| 学习间隔               | 两次学习之间最少隔多久（默认60分钟）                |
| 启用内容守卫             | 拦截 LLM 直接输出的文本，只放行通过工具发送的消息        |
| 防抖间隔（秒）            | 群聊连发时跳过中间消息，只处理最后一条。0=关闭           |
| 注入最近N条消息作为上下文      | 将最近 N 条群聊消息注入 LLM 请求。0=关闭          |
| 保留最近N轮对话           | 裁剪 LLM 上下文保留最近的 N 轮对话。0=全部清除       |
| 全局模型覆盖/学习/选择/审核/推断 | 各任务专用模型，留空用系统默认                    |
| Bot 名称             | 机器人叫什么名字，留空用平台默认名                  |

## 小提示

- 刚装好需要积累一些聊天才会开始学习，耐心等待
- 可以在统计页点击"手动触发学习"来立刻学一次
- 如果学到了奇怪的内容，点拒绝就好
- 如果机器人的回复风格太刻意，可以调低学习触发阈值或关掉自动审核
- 如果 LLM 不使用 `send_message_to_user` 工具，所有回复都会被守卫拦截
