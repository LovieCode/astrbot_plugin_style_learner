# 表达风格学习插件

让 AI 机器人学会群友的说话方式，回复时更有"人味"。

## 功能

**自动学习**：观察群聊，记住常用表达方式和梗，按情绪分类（开心/嘲讽/无奈/鼓励…），注入 LLM 回复风格。

**黑话词典**：自动识别缩写和黑话（yyds、xswl、内卷），推断含义。

**内容守卫**：LLM 只能通过 `send_message_to_user` 工具发送消息，直接输出的"心里话"会被拦截，避免原生文本泄露。

**图片感知**：支持多模态模型直接识别图片；非多模态模型自动生成图片描述供学习使用。描述缓存持久化，重启不丢。

**WebUI 兼容**：新版 AstrBot 使用内嵌管理页面；旧版自动启动自建 Web 管理页（`http://localhost:6187`）。

## 安装

1. AstrBot 插件市场搜索"表达风格学习"，点击安装
2. 或手动放到 `data/plugins/` 下
3. 重启或重载插件

无额外依赖。

## 管理页面

### 新版 AstrBot
打开 WebUI → 侧边栏 → 插件 → 找到本插件 → 点击"页面"。

### 旧版 AstrBot
插件自动在 `http://localhost:6187` 启动自建管理页，含四个标签：
- **统计** — 概览 + 手动触发学习
- **表达** — 按情绪/状态筛选、分页、审核
- **黑话** — 搜索、分页、编辑含义
- **配置** — 可视化修改所有参数

## 指令

在群里发送：

| 指令 | 作用 |
|------|------|
| `/style list` | 查看表达方式 |
| `/style list 开心` | 只看"开心"类 |
| `/style stats` | 学习统计 |

## 配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 启用表达学习 | 开 | 总开关 |
| 启用黑话挖掘 | 开 | 总开关 |
| 注入模式 | append | append=追加到上下文，both=也提供 LLM 工具 |
| 选择模式 | classic | classic=LLM 分析语境，random=随机抽取 |
| 最少学习消息数 | 30 | 攒够后触发一次学习 |
| 学习间隔(分钟) | 60 | 两次学习之间最少间隔 |
| 仅已审核 | 关 | 只使用管理员确认过的表达 |
| 全局黑话/表达 | 关 | 跨群共享 |
| Guard 开关 | 开 | 拦截非工具输出的文本 |
| 防抖(秒) | 1 | 连发时跳过中间消息，只处理最后一条。0=关闭 |
| 平滑(秒) | 2 | 连续 LLM 调用最小间隔。0=关闭 |
| 最大历史轮次 | 1 | 保留最近 N 轮对话。0=全部清除 |
| 最近消息数 | 0 | 注入 N 条最近群聊消息到上下文。0=关闭 |
| 自动审核 | 开 | 定期自动审核新学到的表达 |
| 自动审核间隔(秒) | 300 | |
| 自动审核条数 | 5 | |
| 模型覆盖 | 空 | learn/selection/check/infer 可分别指定模型 |
| Bot 名称 | 空 | 留空用平台默认名 |

## 图片功能

- **多模态模型**：开启后直接传图给 LLM
- **非多模态模型**：需要在 AstrBot 全局配置 `provider_settings.default_image_caption_provider_id` 指向一个支持图片的模型，核心自动生成描述文本替代图片
- 插件端自动缓存图片描述到 SQLite，重启不丢，学习时会注入到上下文中

## 调试

设置日志级别为 DEBUG 后，所有插件日志以 `[SL]` 为前缀，可用 `rg "\[SL\]"` 一键过滤。

## 注意

- 刚装好需积累一定量聊天才会开始学习，可在统计页手动触发
- 学到奇怪内容点拒绝即可，不会影响已通过的内容
- LLM 不使用 `send_message_to_user` 工具时所有回复会被 guard 拦截
